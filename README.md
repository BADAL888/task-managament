# Mission Control Registry (MCR)

A high-performance operational interface designed for directive management and mission oversight. Built on a distributed MERN architecture, this system provides a secure neural grid for operational planning.

## Advanced Features
- **Directive Initialization**: Complex 2-column input system with priority matrixing.
- **Neural Synchronization**: Real-time registry updates via a robust REST interface.
- **Operational Domains**: Categorize directives into Technical, Admin, Research, or Strategy domains.
- **Cyber-Aesthetic Interface**: Premium Charcoal & Emerald theme with Sidebar navigation and ambient visual rendering.
- **Purge Protocols**: Secure deletion of operational data.

## Operational Procedures

### 1. Link Backend
- Navigate to `/backend`
- Initialize packages: `npm install`
- Link MongoDB URI in `.env`.
- Establish connection: `npm run dev`

### 2. Sync Frontend
- Navigate to `/frontend`
- Link assets: `npm install`
- Initialize viewport: `npm run dev`
- Target: `http://localhost:5173`

## Neural API Mapping
| Protocol | Target | Action |
|----------|--------|--------|
| GET | `/api/tasks` | Synchronize registry |
| POST | `/api/tasks` | Initialize new directive |
| PUT | `/api/tasks/:id` | Modify directive state |
| DELETE | `/api/tasks/:id` | Execute purge protocol |
